# rm-github-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that I&#x27;m a bot that automates cloud deployment work for reMarkable

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
